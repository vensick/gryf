Privacy Policy — Gryf Bot
1. Data Collected
Gryf Bot processes only the data required for Discord bot functionality, including:

message content sent directly to bot commands,

user IDs,

server IDs,

channel IDs,

interaction metadata.

The bot does not store or log personal information outside Discord’s infrastructure.

2. Data Storage
Gryf Bot does not store data persistently.
All processing happens in real time and is not saved to external databases.

3. Third Parties
The bot does not share any data with third parties.
All communication occurs exclusively through Discord’s API.

4. User Rights
If you wish to remove your data, simply remove the bot from your server.
No additional action is required because the bot does not store data externally.

5. Contact
For questions or concerns, contact the bot owner via GitHub:
https://github.com/vensick/gryf
